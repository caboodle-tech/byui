Design Principles

In this activity we will introduce you to some basic design principles. These principles are core to design and can be applied to any kind of design work you might find yourself asked to do in the future. As you visit websites, view posters, read magazines, watch movies, and so on over the next few days, please look out for examples of these principles being used. If you watch for them, you will start seeing examples of these design principles everywhere.

  2. Please review and consider keeping a copy of the following info-graphics:

    • Elements of Design
