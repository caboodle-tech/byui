  2. Read the Resource

    CSS-Tricks is an awesome tech blog that started its life posting about CSS tricks for cross-browser design. In the past various web browsers often displayed the same web designs in very different ways. Developers and designers had to add extra code to insure their designs looked and operated the same cross-browser. This problem, well substantially reduced today, still exists. The CSS-Tricks website has evolved to cover many topics and I would recommend them to you as another resource for software development and web design.

    Please read CSS-Tricks article Does CSS Grid Replace Flexbox? which helps compare and contrast flexbox and grid.
